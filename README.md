# mp_moneywash
QB Moneywash Script by MindlessPuppetz

## Dependencies:
* [QBCore Framework](https://github.com/qbcore-framework)
* [Ox Lib](https://github.com/overextended/ox_lib/releases)

## Installation

* Download ZIP
* Drag and drop into `[standalone]` remove `-main` from file name
* Restart your server
